User Guide

🚀 Quick Start

Step 1: Install Dependencies

Open terminal in the project directory and run:

pip install -r requirements.txt

Step 2: Run Simulation

Execute the simulation:

python main.py

Step 3: View Dashboard

Open in your browser:

report/index.html

🔄 How the Dashboard Works

💡
Dynamic Data Loading The dashboard is a static HTML page that loads data from results/latest.json. Every time you run the simulation, the latest JSON file is updated. Refresh the browser to see new results.

⌨️ Command Line Options

Flag Values Default Description
--tasks Integer (N) 10 Number of tasks to generate
--grid Integer (N) 20 Grid size (NxN)
--out Path (auto) Output JSON path (saved under results/)
--seed Integer (N) 42 Random seed for reproducibility
--no-delay Flag Off Skip execution delays (faster demo)
--interactive Flag Off Open interactive menu to edit settings and run multiple simulations
--out Path string (auto) Custom output JSON path — auto-generates t{n}_g{g}_s{s}_*.json if omitted

Basic Examples

Quick Demo

python main.py --no-delay

More Tasks

python main.py --tasks 20 --seed 123

Custom Grid

python main.py --grid 30

Interactive Mode

python main.py --interactive

🔧 Troubleshooting

❌ "Could not load simulation data"

The dashboard cannot find results/latest.json.
Solution: Run python main.py first, then refresh the page.

❌ "ModuleNotFoundError"

Python dependencies are not installed.
Solution: Run pip install -r requirements.txt

❌ "ModuleNotFoundError: No module named 'tabulate'"

A required dependency is missing.
Solution: Run pip install tabulate colorama

❌ Dashboard not updating

Browser may be caching old data.
Solution: Hard refresh (Ctrl+Shift+R) or clear browser cache.
